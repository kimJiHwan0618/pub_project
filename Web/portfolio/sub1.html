삼성전기

해당 사이트는 웹 콘텐츠 접근성 지침에 따라 제작하였으며,
W3C 유효성 검사를 통과하였습니다.

1. W3C 유효성 검사

W3C HTML 유효성 검사와 CSS 유효성 검사를 통과하고
웹 표준에 맞게 개발되었다는 사실을 증명하는 인증마크를 획득했습니다.

2. 웹 접근성

한국형 웹 콘텐츠 접근성 지침2.0을 평가기준으로 하여
준수율 100%로 웹 접근성 검사를 통과했습니다.

3. Cross Browsing

IE8 - IE11, Chorme, FireFox, Opera, Safari에서
동일한 웹페이지를 볼 수 있습니다.
